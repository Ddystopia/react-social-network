## About project

This is a social network, and have these abilities:
- login
- logout
- See another users and their profiles
- Make profile
	- Set photo
	- Set about me
	- Set status
	- Set looking for a job
	- Set constants
- Create new chats
- Send messages
- Receive messages 
- Remove messages
- Check message was viewed
